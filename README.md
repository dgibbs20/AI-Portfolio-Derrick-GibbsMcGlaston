## Smart Budgeting Agent (Capstone Project)

A full-stack AI application that categorizes financial transactions using a machine learning model built with Scikit-Learn.

Features include:
- SVC-based ML model with ~90% accuracy  
- FastAPI backend  
- SQLite database  
- Clean HTML/JS frontend  
- Real-time category prediction  
- Modular Python codebase  

All source code is included in the `Phase05_Final_Project/` folder.
